# fgeeha.github.io

Статический сайт-портфолио и набор небольших одностраничных инструментов и игр, публикуемых через [GitHub Pages](https://pages.github.com/).

## Уже в репозитории

| Папка | Назначение |
|--------|------------|
| Корень `index.html` | Главная страница, навигация к мини-проектам |
| `infinity` | Интерактивная визуальная анимация |
| `PasswordGenerator` | Генератор паролей с учётом энтропии |
| `GreenSentry` | Лендинг / концепт ИИ-диагностики растений |
| `csvEditor` | Редактирование CSV в браузере |
| `FileEncodingIdentifier` | Определение кодировки файла |
| `DungeonDash`, `DoomDash` | Игры на canvas |
| `WorkDayTimer` | Таймер рабочего дня |
| `Roulette` | Рулетка с персонажами (можно добавить в меню главной при желании) |
| `Candle`, `Candles`, `YouTube` | Другие страницы / эксперименты |

Ниже идеи в духе уже существующих страниц: без сервера, максимум ванильный JS или лёгкие CDN-библиотеки, удобно для GitHub Pages.

### Инструменты и утилиты

- [ ] **JSON / YAML форматтер** — вставка «кривого» JSON, prettify, проверка синтаксиса, копирование в буфер. Полезно рядом с CSV-редактором для людей, кто крутит конфиги и API.
- [ ] **Base64 и простые кодировки** — текст в Base64 и обратно, опционально URL-safe вариант. Можно добавить hex и процент-encoding для URL. Всё на клиенте, без отправки данных.
- [ ] **Генератор UUID / случайных id** — кнопка «сгенерировать N штук», v4 UUID, опционально короткие id (nanoid-стиль на чистом JS). Удобно для разработчиков.
- [ ] **Проверка контраста цветов (WCAG)** — два color picker, вывод коэффициента контраста и уровня AA/AAA для текста и крупного текста. Связано с доступностью, хорошо смотрится в портфолио.
- [ ] **Живой предпросмотр Open Graph** — поля title, description, image URL и макет «как в соцсетях». Помогает отлаживать мета-теги без деплоя.
- [ ] **QR-код генератор** — ввод текста или URL, отрисовка QR на canvas или через лёгкую библиотеку, скачать PNG. Частый запрос, реализация компактная.
- [ ] **Сравнение двух текстов (diff)** — два textarea, построчное или посимвольное выделение отличий. Полезно для конфигов, логов, мелких правок.

### Обучение и «игрушки»

- [ ] **Игра «Жизнь» Конвея** — сетка, стартовые пресеты, шаг и автозапуск, подсчёт поколений. Классика для canvas и логики на чистом JS.
- [ ] **Тест скорости печати** — случайная цитата на русском/английском, таймер, WPM/CPM, опечатки. Минимум графики, упор на UX клавиатуры.
- [ ] **Визуализатор сортировок** — столбики массива, пошаговая анимация пузырька, быстрой сортировки, слиянием. Хорошо для демонстрации алгоритмов.
- [ ] **Часы по часовым поясам** — выбор 3–6 городов, одна шкала времени, учёт локального времени браузера. Дополняет Work Day Timer с другой стороны.

### Медиа и мелочи

- [ ] **Сжатие картинки в браузере** — загрузка JPG/PNG, ползунок качества, вывод через canvas в WebP/JPEG, скачать файл. Без сервера, с оговоркой про размер файлов в памяти.
- [ ] **Простой графический редактор** — холст, кисть, ластик, цвет, очистить, сохранить PNG. Можно как «скетчборд» без сохранения на сервер.

## Локальная проверка

Откройте `index.html` в браузере или поднимите статический сервер в корне репозитория (например `python -m http.server`), чтобы корректно отрабатывали относительные пути к подпапкам.

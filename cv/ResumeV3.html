Lenta Tech Life Hack 2026

Команда «Стабилизируй это» · Капитан, ML/CV Engineer, MLOps

Разработал ML/CV-пайплайн для автоматического распознавания ценников с видеопотока робота-инвентаризатора. На входе — 4K-видео полок гипермаркета, на выходе — структурированный CSV из 29 полей по каждому ценнику (цены, штрихкод, артикул, дата печати, bbox-координаты). Стартовали с 0/274 распознанных и за счёт архитектурных решений довели метрику до 47/274 (0.1715).

0.1715 ALL_VALUE метрика
175/175 unit-тестов прошли

  • Спроектировал пайплайн детекции и трекинга: YOLOv8n tiled (4K → тайлы 640×640, mAP50 = 0.776), MSER fallback, ByteTrack и per-video lens correction по официальной калибровке камеры.
  • Реализовал каскад декодеров QR/штрихкодов: WeChat QR → pyzbar → cv2.barcode → zxing-cpp → qreader с валидацией и репаром EAN-13.
  • Построил OCR-стек: PaddleOCR для чисел и цен, EasyOCR ru для названий товаров; layout-парсер, нормализация цен, дат и кодов.
  • Внедрил 4-уровневый catalog lookup с пропагацией полей через штрихкод и fuzzy-match по уникальным словам — главный breakthrough 0/274 → 35/274.
  • Настроил CI/CD на GitHub Actions: ruff lint, 175 unit-тестов, авто-деплой на HuggingFace Spaces только при зелёных тестах. Собрал Docker-образ для production-деплоя.
  • Сделал Gradio-демо для запуска пайплайна через веб-интерфейс и интерактивной проверки результатов.

Настройка RAG для Q&A

Альфа-Будущее · Changellenge 2025

Команда «MireaFive» · ML Engineer, RAG, NLP · Ноябрь 2025

Разработал RAG-систему для поиска релевантных веб-страниц по пользовательским вопросам на русском языке. На входе — корпус из ~7 000 веб-документов и набор вопросов, на выходе — ранжированный топ-5 наиболее релевантных страниц на каждый запрос. Стартовали с нуля и поздно включились, но итеративно вышли в верхнюю половину лидерборда.

0.34 → 0.42 F1 · публичный LB
8/день лимит сабмитов
RTX 4090 локальное железо
  • Построил гибридный ретривер: BGE-M3 эмбеддинги + BM25 с RRF-фьюжином для объединения лексического и семантического поиска.
  • Реализовал Query Expansion через Qwen3-30B: для каждого вопроса генерировались 2–3 парафразы, результаты поиска объединялись через RRF — дало первый заметный прирост метрики.
  • Экспериментировал с HyDE (Hypothetical Document Embeddings) и Convex Combination fusion для взвешенного объединения BGE-M3, BM25 и HyDE-запросов; из-за нестабильности и лимита сабмитов не удалось полностью отладить.
  • Подключил cross-encoder реранкер с 4-битной квантизацией: чанки по 700 токенов вместо 2000 — компромисс под 4090; реранкинг прошёл за 8.5 ч.
  • Провёл предобработку корпуса: чистка HTML, нормализация пробелов и lowercase; намеренно отказался от стемминга и стоп-слов — изменения текста вопросов роняли метрику.
  • Итерировал по embedding-моделям (paraphrase-multilingual-mpnet, BGE-M3, Qwen 8B) и параметрам чанкинга — выявил, что поиск без чанкинга давал максимальный базовый скор.
  • Финал — выбыли по reshuffle: организаторы пересчитали приватный LB с ограничением «топ-5», обнулив команды, которые отправляли >5 ссылок как «хак». Команда уроки усвоила и подготовилась к следующему раунду.
